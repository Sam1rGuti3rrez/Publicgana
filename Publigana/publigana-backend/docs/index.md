# Publigana Backend Documentation

Bienvenido a la documentacion oficial de Publigana Backend.

## Objetivos del sitio

- Definir la arquitectura tecnica objetivo del proyecto.
- Guiar a desarrolladores en practicas de implementacion, seguridad y calidad.
- Servir como referencia estable para evolutivos y mantenimiento.

## Alcance

Este sitio cubre:

- Fundamentos de backend empresarial
- Java 21 y Spring Boot 4
- Spring Security, JPA, Hibernate y PostgreSQL
- Clean Architecture, Hexagonal, SOLID y Clean Code
- Testing, observabilidad, despliegue e infraestructura
- Aplicacion practica al dominio Publigana
